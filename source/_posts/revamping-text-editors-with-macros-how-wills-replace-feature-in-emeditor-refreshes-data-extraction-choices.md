Reproduction:  
 1\. Set Extract Options to “Display Matched Lines Only” or “Display Matched Strings Only”  
 2\. Start recording (Macro)  
 3\. Find X  
 4\. Replace with Y  
 5\. Stop recording (**Extract Options no change**)  
 6\. Run Macro (**Extract Options no change**)  
 7\. Macros -> Save as : “test.jsee”  
 8\. Macros -> Select “test.jsee”  
 9\. Run Macro test.jsee (**Extract Options has been reset to “Display File Names and Lines”**)  
 10\. Or click Macros Toolbar “test.jsee” (**Extract Options has been reset to “Display File Names and Lines”**)
